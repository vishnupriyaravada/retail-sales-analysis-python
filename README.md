# Retail Sales Data Analysis

## Project Overview
This project analyzes a retail sales dataset using Python to uncover sales trends, product performance, and profitability patterns.

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  

## Dataset
The dataset contains retail sales records including product category, region, sales, profit, discount, and customer segment.

## Analysis Performed
• Sales by category  
• Profit by region  
• Sales distribution by customer segment  
• Top performing sub-categories  
• Discount vs profit relationship  

## Key Insights
• Technology category generates the highest revenue  
• West region contributes the highest profit  
• Consumer segment accounts for the largest share of sales  
• Higher discounts often reduce profit margins
